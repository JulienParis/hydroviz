HYDROVIZ est une application expérimentale et certaines fonctionnalités sont encore en cours de développement. Le code est en open source sur GitLab. Les données et les fonds de carte ont été fournis par le ministère de l'environnement, de l'énergie et de la mer dans le cadre du concours de data-visualisation sur les pesticides dans les eaux souterraines (voir également le règlement du concours ).
